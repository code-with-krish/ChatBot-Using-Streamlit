# ChatBot Using Streamlit

This is a simple chatbot web application built using **Streamlit** and Python.  
The chatbot can respond to user inputs and provide interactive conversations in a web interface.

## Features

- Chat with a simple AI-based chatbot
- Built with Python and Streamlit
- Easy to run locally
- Clean web interface

## Installation

1. Clone this repository:

```bash
git clone https://github.com/code-with-krish/ChatBot-Using-Streamlit.git
```
